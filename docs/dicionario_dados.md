# Dicionário de Dados

## orders.csv
Tabela contendo informações dos pedidos realizados na plataforma.

## order_items.csv
Tabela com o detalhamento dos itens de cada pedido.

## products.csv
Catálogo de produtos disponíveis no e-commerce.

## users.csv
Informações dos usuários/clientes.

## events.csv
Eventos de navegação e interação dos usuários na plataforma.
